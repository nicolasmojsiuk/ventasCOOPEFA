# VentaFácil

VentaFácil es un sistema de gestión web desarrollado como proyecto personal, construido con Django.
El objetivo del proyecto es crear una solución flexible para la administración de un comercio, centralizando la información y automatizando los procesos principales.

El sistema se encuentra actualmente en desarrollo y se irá ampliando de forma progresiva.

---

## 🎯 Objetivo del proyecto
Desarrollar un sistema de gestión web que permita administrar de manera eficiente las operaciones internas de un comercio, aplicando buenas prácticas de desarrollo backend con Django.

---

## ⚙️ Funcionalidades previstas
El proyecto tiene como objetivo implementar los siguientes módulos:

- **Gestión de empleados**
  - Administración de usuarios internos.
  - Control de accesos y roles.

- **Gestión de caja**
  - Registro de ingresos y egresos.
  - Control diario de movimientos de efectivo.

- **Gestión de ventas**
  - Registro de operaciones de venta.
  - Asociación de productos y clientes.

- **Presupuestos**
  - Creación de presupuestos para clientes.
  - Seguimiento del estado de cada presupuesto.

- **Informes**
  - Reportes de ventas y movimientos.
  - Análisis básico para la toma de decisiones.

- **Gestión de inventario**
  - Administración de productos.
  - Control de stock disponible.

- **Gestión de clientes**
  - Registro y seguimiento de clientes.
  - Historial de operaciones.
